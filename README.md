# AURA – Smart EdTech & Career Growth Platform

AURA is a modern full-stack EdTech platform designed to help students learn, grow, and get career-ready through premium courses, AI assistance, projects, and placement-focused tools.

Developed and maintained by **Anil Ganjivarapu**

---

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Responsive UI / Premium Design

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT Token Authentication
- bcrypt Password Encryption
- Role-Based Access

### AI Integration
- Google Gemini API
- OpenAI Ready Architecture

### Payments
- Razorpay Integration Ready

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
